# Portfolio Website

A modern portfolio website showcasing AI and web development projects. Built with HTML, CSS, Tailwind, and vanilla JavaScript.

## Features

- **Responsive Design** - Mobile-first, works on all devices
- **Modern Tech Stack** - Tailwind CSS, Anime.js, Typed.js animations
- **Project Showcase** - Interactive project cards with live demos
- **Skills Section** - Organized technical skill categories
- **Blog Integration** - Dynamic blog with posts
- **Contact Page** - Email obfuscation and social links

## Quick Start

```bash
git clone https://github.com/joeynyc/terminalos.git
cd terminalos
open index.html
```

## Tech Stack

- **Core**: HTML5, CSS3, Vanilla JavaScript
- **Styling**: Tailwind CSS (CDN)
- **Animations**: Anime.js, Typed.js, Splitting.js
- **Fonts**: Inter, JetBrains Mono

## Project Structure

```

├── blog.html          # Blog page
├── contact.html       # Contact page
```
