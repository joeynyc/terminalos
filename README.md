## ✨ Overview

TerminalOS revolutionizes the portfolio experience by creating an authentic terminal environment in your browser. Built with vanilla JavaScript for maximum performance, this interactive portfolio features over 20 custom commands, real-time RSS integration, hidden easter eggs, and a complete command-line interface that showcases professional skills while entertaining visitors.

### 🎯 What Makes This Special?

- **🎨 Authentic Terminal UX** - Real command history, tab completion, and terminal behaviors
- **📱 Fully Responsive** - Seamless experience across desktop, tablet, and mobile devices
- **⚡ Zero Dependencies** - Pure vanilla JavaScript for blazing-fast performance
- **🔐 Security-First** - Comprehensive CSP implementation with external API integration
- **🎪 Interactive Portfolio** - Gamified exploration of skills, projects, and experience
- **📰 Live Data Integration** - Real-time Hacker News RSS feed with smart error handling

## 🚀 Quick Start

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software required!

### Installation Options

#### Option 1: Direct Download
```bash
# Clone the repository
git clone https://github.com/joeynyc/terminalos.git
cd terminalos

# Open in browser
open index.html
# or simply double-click index.html
```

#### Option 2: Live Server (Recommended for Development)
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx serve .

# Using PHP
php -S localhost:8000
```

#### Option 3: Deploy to GitHub Pages
1. Fork this repository
2. Go to Settings → Pages
3. Select "Deploy from a branch" → `main` → `/ (root)`
4. Your site will be live at `https://yourusername.github.io/terminalos/`

## 🎯 Commands Reference

<div align="center">

### 📋 **Quick Command Guide**

</div>

#### 👤 Portfolio Commands
```bash
about_me        # Personal introduction and background
show_tools      # Technical skills and expertise
skills          # Interactive skills visualization
projects        # Portfolio showcase with live demos
resume          # Professional experience and education
contact_joey    # Contact information and social links
github          # Open GitHub profile
linkedin        # Open LinkedIn profile
```

#### 🖥️ System Commands
```bash
help            # Show all available commands
clear           # Clear terminal screen
ls              # List directory contents
pwd             # Show current directory path
whoami          # Display current user information
date            # Show current date and time
uname           # System information display
cat [file]      # Display file contents
echo [text]     # Echo text to terminal
```

#### 🎭 Special Commands
```bash
sudo su         # Elevate to root user (unlocks secrets)
secret          # Access classified content (root only)
matrix          # Enter matrix mode with effects
hack            # Initiate hack simulation sequence
rss             # Live Hacker News feed integration
exit            # Attempt system exit (with surprises)
```

#### 💡 Pro Tips
- Use **Tab** for command autocompletion
- Navigate history with **↑/↓** arrow keys
- Try `help [command]` for detailed command info
- Discover hidden easter eggs by exploring!

## 🏗️ Architecture

### 🗂️ Project Structure
```
terminalos/
├── 🏠 index.html              # Main terminal interface
├── ⚡ script.js               # Core terminal engine (73KB)
├── 🎨 styles.css              # Visual styling and animations
├── 📱 mobile-test.html        # Mobile testing utilities
├── 📝 blog.html               # Blog integration
├── 📋 projects.html           # Project showcase page
├── 🔧 tools/                  # Utility applications
│   ├── focusclock.html        # Productivity timer
│   └── git-cheatsheet.html    # Git reference guide
├── 🌐 browseros/              # Web-based OS simulation
├── 💰 cryptowatch/            # Cryptocurrency tracker
├── 🧠 expneural/              # Neural network visualization
├── 🖼️ svgviewer/              # SVG gallery and editor
├── 📊 images/                 # Static assets
├── 🎬 videos/                 # Demo videos and media
└── 📄 posts.json              # Blog post data
```

### 🔧 Core Architecture Components

#### Terminal Engine (`script.js`)
- **Command Parser** - Processes and validates user input
- **History Manager** - Maintains command history and navigation
- **Output Renderer** - Handles terminal output and formatting
- **State Manager** - Manages user sessions and preferences
- **API Integration** - Handles external data fetching

#### Security Layer
- **Content Security Policy** - Prevents XSS and code injection
- **CORS Proxy System** - Secure external API access
- **Input Sanitization** - Validates and cleans user input
- **Error Boundaries** - Graceful error handling and recovery

#### Mobile Optimization
- **Touch Event Handling** - Mobile-specific interactions
- **Responsive Breakpoints** - Adaptive layout system
- **Performance Optimization** - Mobile-first loading strategies
